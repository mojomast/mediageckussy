# Roadmap And Limitations

## Complete In MVP
- TV-series format pack
- Feature-film format pack
- Podcast format pack
- Web-series format pack
- Canon YAML model with field metadata
- Manifest-driven package generation
- Partial regeneration by department or file
- Static public website export
- Basic validation and stale-output warnings
- Example canon and generated package
- Deploy-folder publishing helper

## Stubbed Or Deferred
- Full support for game, book/comic, and album packages
- Partner bundle and press bundle archive exports
- Protected manual-edit blocks
- Rich dependency graph visualization
- Guided intake wizard or web UI
- Advanced editorial analysis such as tone drift

## Next Useful Additions
1. Add format packs for game and book/comic.
2. Introduce protected sections for human-owned content.
3. Add bundle export commands and redaction rules.
4. Persist generation metadata separately from generated files.
